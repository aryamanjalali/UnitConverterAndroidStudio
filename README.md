# Unit Converter Android App

A comprehensive Android application for converting units across multiple measurement categories. This app provides an intuitive user interface with real-time conversion calculations for temperature, length, and number systems.

## Overview

Unit Converter is a feature-rich Android application that supports multiple unit conversion categories. The app features a modern Material Design interface, real-time calculations, and error handling for a seamless user experience.

## Features

- **Temperature Conversion**: Convert between Celsius, Fahrenheit, and Kelvin
- **Length Conversion**: Convert between various length units (meters, feet, inches, etc.)
- **Number Conversion**: Convert between different number systems (binary, decimal, hexadecimal, octal)
- **Real-time Calculations**: Instant conversion as you type
- **Error Handling**: Input validation and error messages
- **Modern UI**: Material Design with navigation drawer
- **Responsive Design**: Works on various Android device sizes

## Technologies Used

- **Java**: Core programming language
- **Android SDK**: Android development framework
- **Gradle**: Build automation
- **Material Design**: UI components

## Installation

1. Clone the repository:
```bash
git clone https://github.com/aryamanjalali/UnitConverterAndroidStudio.git
cd UnitConverterAndroidStudio
```

2. Open the project in Android Studio

3. Sync Gradle dependencies

4. Build and run on an Android device or emulator

## Usage

1. Launch the app on your Android device
2. Select a conversion category from the navigation drawer:
   - Temperature
   - Length
   - Number
3. Enter the value to convert
4. Select source and target units
5. View the converted result in real-time
